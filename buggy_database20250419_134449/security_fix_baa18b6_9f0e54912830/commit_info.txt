Full commit message:
platform/x86: hp-bioscfg: Replace deprecated strncpy() with strscpy()

Since kstrtol() requires a NUL-terminated string as input and strncpy()
is deprecated for NUL-terminated destination buffers, use strscpy()
instead.

Compile-tested only.
